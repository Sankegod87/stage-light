NFC Tags

Tap-to-scan tags for fixtures
📲 Option 1 — URL Tags (iPhone + Android)

Program NFC sticker tags with a URL. When tapped, the phone opens your app and auto-loads the fixture. Works on both iPhone and Android without any special setup.

Buy: Search "NFC sticker tags NTAG213" on Amazon — ~$10 for 30 tags.

Program your NFC tags — one per fixture
📡 Option 2 — Web NFC (Android Chrome only)

Use your phone to write NFC tags directly from this app and read them without opening a URL. Android Chrome only — does not work on iPhone.
